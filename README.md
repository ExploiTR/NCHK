• Dump network downtimes with time stamp.

#### Extra Infos :
• UI is desgined with IntelliJ Idea. (GUI Form - GridLayoutManager)  
• `Not X-Platform!` - As the application relays on ICMP Ping, the ping command is different in different OS.
